# pokemon-party
Pokémon Party Tool to display to your livestream the Pokémon in your party.

### Prerequisites

* [NodeJS](https://nodejs.org/en/) - NodeJS v.8.7.0

### Installing

Run npm install to download dependencies.
```
npm install
```

Run npm start to start the server
```
npm start
```

View in browser
```
http://localhost:4200/
```
